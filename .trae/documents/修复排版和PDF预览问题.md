# 修复排版和PDF预览问题

## 问题1：排版问题

### 1.1 左侧分页区域固定高度
**当前问题**：左侧列表区域没有固定高度，会随着内容增长
**解决方案**：
- 给左侧边栏设置固定高度（如 `h-[calc(100vh-2rem)]`）
- 确保列表区域有独立滚动

### 1.2 版本历史改为下拉按钮
**当前问题**：水平卡片布局导致版本多时预览区域被挤压
**解决方案**：
- 将版本历史改为一个按钮，显示当前版本号
- 点击按钮展开下拉列表，选择版本后自动收起
- 下拉列表使用绝对定位，不影响其他元素布局

## 问题2：PDF预览崩溃

### 2.1 移除react-file-viewer
**当前问题**：react-file-viewer组件导致PDF预览崩溃
**解决方案**：
- 完全移除PDF预览功能
- PDF文件显示友好提示："PDF暂不支持预览，请下载后查看"
- 保留下载按钮功能

### 2.2 清理依赖
- 从package.json移除react-file-viewer依赖
- 简化代码，移除相关导入

## 具体修改

### DocRepository.tsx修改
1. 左侧边栏：添加固定高度类
2. 版本历史：改为下拉按钮 + 下拉列表
3. PDF预览：移除FileViewer，显示提示信息
4. 清理导入：移除react-file-viewer相关代码

### package.json修改
- 移除 "react-file-viewer": "^1.2.1"
